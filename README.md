# SME-Financial-Distress-Tableau

## Project Overview
This project analyzes the factors associated with financial distress in SMEs (Small and Medium Enterprises).

## Key Findings
- Financial distress varies across SME characteristics, especially firm size
- Risk management shows only a weak relationship with distress
- Financial literacy and decision-making have the strongest impact on distress

## Tools Used
- Tableau (Dashboards & Story)
- Excel (Data preparation)
